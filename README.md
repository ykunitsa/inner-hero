# Inner Hero - Приложение для КПТ Экспозиций

Приложение для упражнения "Экспозиция" из когнитивно-поведенческой терапии.

## Технологии
- iOS 17+
- SwiftUI
- SwiftData
- Локальное хранилище (без авторизации)

## Структура проекта

### Модели
- **Exposure.swift** - модель данных для экспозиций
  - `title` - название экспозиции
  - `situation` - описание ситуации
  - `fearLevel` - уровень страха (1-10)
  - `createdAt` - дата создания
  - `completedSessions` - количество пройденных сеансов

### Экраны

#### 1. Вкладка "Экспозиции" (ExposuresListView)
- Список всех созданных экспозиций
- Возможность добавления новых экспозиций
- Отображение уровня страха и количества сеансов
- Удаление экспозиций

#### 2. Вкладка "Сеанс" (SessionView)
- Режим прохождения экспозиционной терапии
- Базовый UI для начала сеанса
- (Готово к дальнейшей разработке)

## Запуск проекта

1. Откройте `Inner Hero.xcodeproj` в Xcode
2. Выберите симулятор iOS 17+ или физическое устройство
3. Нажмите Cmd+R для запуска
